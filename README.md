# Hydroponics Dashboard

Real-time hydroponics monitoring using Arduino Uno and Streamlit.

## Features
- Read TDS, pH, Water Level, Light, and Temperature in real-time
- Predict NPK levels using ML model
- Display comparison and difference between predicted and real-time
- Sidebar for plant type and growth stage
- Live metric cards and alerts

## Requirements
- Python 3.10+
- Libraries: streamlit, pandas, scikit-learn, pyserial, joblib

## Run
```bash
streamlit run streamlit_app.py
